# Maize-NIRS-GBS #

This repository is intended to serve as a host for supplementary materials associated with the manuscript titled 
"Near Infrared Reflectance Spectroscopy Phenomic and Genomic Prediction of Maize Agronomic and Composition Traits Across Environments".

The file 20230719_BGLR_Prediction_Script_V23.R contains all code necessary to run genomic, phenomic, and combined genomic/phenomic prediction.

To run the script, create a new folder and save the R script within it. Then unzip the contents of CSVs.zip and SNP60000.hmp.zip and save
these files into the folder containing the R script. Once the R script has been opened, set the working directory to the folder that 
contains the R script, .hmp file, and all of the .csv files.

Supplementary_Data_1.xlsx is a spreadsheet that contains prediction abilities for across- and within-environment predictions,
results from GWAS, and variable importance scores assigned by the lasso algorithm for each NIRS band.

Supplementary_Figures.pdf contains supplemental figures mentioned in the manuscript.
